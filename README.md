# Contrast Lab

Contrast Lab is a small benchmark and playground for exploring how accessibility tools behave on cases related to **WCAG 2.1 Success Criterion 1.4.3 (Contrast Minimum)**. The site includes ready-made examples, from simple colour combinations to more complex cases with gradients, images, overlays, filters, and layered elements. You can inspect the existing cases, modify them, or add new ones to test a specific situation.

The main page lets you browse the benchmark, filter cases, and record tool results in a structured way. You can search by case or tag, focus on the **core** set for more straightforward examples or the **challenge** set for cases that are harder for tools to interpret, and export the completed session as a structured CSV file for later analysis. Tool output is **not collected automatically**: you run the tool yourself and manually note what it reports for each case.

If you prefer to inspect one scenario at a time, each case also has its own dedicated page. This is useful for tools that work page by page or when you want to focus on a single example without the full benchmark view.
